# Germany Renewable Capacity Analysis

This project analyzes Germany's projected renewable electricity capacity and electricity demand from 2030 to 2040 using R.

## Files
- `germany_renewable_outlook.Rmd` – full analysis code
- `germany_renewable_outlook.pdf` – final report
- `future_agorameter.xlsx` – raw dataset

## Main Insights
- Renewable capacity grows faster than electricity demand.
- Solar (photovoltaic) is the largest growth driver.
- Wind remains an important contributor.
- Capacity growth alone does not guarantee supply adequacy.

## Tools Used
R, tidyverse, ggplot2, readxl
